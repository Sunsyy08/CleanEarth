# CLEAN EARTH

분리배출이 가능한 물건을 한 번 쓰고 버려 불필요한 폐기물이 늘어나고 있습니다. 그래서 저희는 카메라로 촬영한 쓰레기의 종류를 인식하고, 그에 맞는 리폼 아이디어를 추천해주는 재활용 리폼 앱 CleanEarth를 제작하게 되었습니다.

* 플랫폼 : 안드로이드/모바일
* 제작 인원 : 4인 (프론트 1명, 백엔드 2명, AI 1명)
* 역할 : 앱 개발 및 UI 디자인
* 제작 기간 : 9일 2025.05.22 ~ 2025.05.30
* 사용 기술 : Kotlin, Compose

### 기능
* 로그인, 회원가입
* 키워드 검색 후 리폼 아이디어 추천
* 이미지 촬영으로 AI가 리폼 가능 여부 판단 후 리폼 추천
* 프로필 화면
